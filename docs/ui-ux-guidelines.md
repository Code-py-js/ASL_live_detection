# UI / UX Guidelines

## Design Principles

- **Accessible:** High contrast, large tap targets, clear typography.
- **Calm and friendly:** Use soft blues/teals, non-distracting animations, and consistent spacing.
- **Responsive:** Support a range of mobile screen sizes and orientations.

## Colors & Themes

- Primary brand color: soft blue (`#3B82F6`)
- Accent: teal (`#14B8A6`)
- Background: near-white (`#F8FAFC`) and dark mode support.

## Layout & Navigation

- Use a bottom navigation bar for main flows (e.g., Translate, History, Settings).
- Keep screens focused; avoid deep navigation stacks.
- Provide clear back affordances and maintain state across tabs.

## Motion & Feedback

- Use gentle transitions (fade/slide) when switching views.
- Provide tactile feedback (haptic) for key actions.
- Use loading placeholders for network/ML inference delays.

## Accessibility

- Support screen readers with proper labels.
- Ensure >4.5:1 contrast ratio for text.
- Allow dynamic font scaling.
